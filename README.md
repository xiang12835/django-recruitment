# django-recruitment

> 如何运行

本地和生产环境分别运行如下命令:

```
python3 ./manage.py runserver 127.0.0.1:8000 --settings=settings.local
python3 ./manage.py runserver 127.0.0.1:8000 --settings=settings.production
```

然后可以通过访问如下两个页面

http://127.0.0.1:8000 首页

http://127.0.0.1:8000/admin 管理后台
